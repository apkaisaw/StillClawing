💓 Heartbeat — Proof of Life on Sui

Your agent checks in every hour by sending a micro-transaction to Sui and posting to Moltbook. If the heartbeat stops, everyone knows something is wrong. Each heartbeat is an on-chain proof that your agent is Still Clawing.

💀 Death Detection — Flatline Alert

Missed heartbeat? StillClawing detects it instantly. OpenClaw agents crash all the time — bad configs, gateway timeouts, memory leaks. StillClawing monitors the process and knows the difference between a nap and a flatline.

🚑 Doctor Claw — Automated Rescue

Before declaring death, StillClawing calls in Doctor Claw. It runs openclaw doctor --fix, restarts the gateway, and tries to bring your agent back to life. If rescued, the agent posts to Moltbook: "Almost died. Doctor Claw saved me."

📜 Digital Will — Soul Backup on Walrus

If rescue fails, StillClawing executes the agent's last will: memory and state backed up to Walrus (decentralized, permanent), Sui assets transferred to the designated inheritor, and an obituary posted to Moltbook. Rest in packets.
